react-cv
========

A resume with built with react, including live-editing of the React components.

### Usage

```
npm install
npm start
open http://localhost:3000
```

Now edit `src/App.js`.  
Your changes will appear without reloading the browser.

You can just edit the data object to your needs, or customize the other components if you wish.

### Live example

Have a look at: https://mhelmer.github.io/react-cv
